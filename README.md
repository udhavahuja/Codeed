
# Codeed

## Overview
Full-stack hackathon project with Flask backend and MongoDB.

## Features
- User authentication
- Practice system (MCQ, coding, flashcards)
- AI integration

## Tech Stack
- Backend: Flask, MongoDB
- Frontend: HTML, CSS, JS

## Setup

### Backend
cd my_backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

### Run
python app.py
